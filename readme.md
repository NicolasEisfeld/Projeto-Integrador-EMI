# MEG: Método de Ensino Gamificado 

O projeto “MEG: método de ensino gamificado para o ensino de ciências da natureza” propõe a criação de uma plataforma gamificada e voltada para auxiliar estudantes do 9° ano no preparo para o processo seletivo de ingresso no IFRS. O objetivo geral do projeto é desenvolver um site com elementos de gamificação, como rankings, moedas e personalização de perfil que auxilie os estudantes nos estudos dessa área, melhorando o desempenho no processo seletivo. A plataforma terá questões e simulados, ambos terão resolução explicada, além da seleção de questões anteriores. 

## Objetivos principais
- Desenvolver uma plataforma online que auxilie no processo de aprendizagem aos estudantes ingressantes;
- Organizar materiais pedagógicos para a plataforma, como vídeo aulas e textos com a supervisão de um professor especialista em cada área;
- Realizar a escrita do relatório detalhando todos os procedimentos realizados ao longo do desenvolvimento do projeto;
